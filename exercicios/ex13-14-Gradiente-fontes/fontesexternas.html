Fontes Externas

Web servers

Importação de fontes online:


            @import url('https://fonts.googleapis.com/css2?
            family=Honk&family=Work+Sans:ital,wght@0,100..
            900;1,100..900&display=swap');

  • @import → carrega estilos externos.
  • Aqui são importadas duas fontes do Google Fonts:
    • Honk
    • Work Sans (com suporte a itálico e variações de peso entre 100–900).

Aplicação das fontes


            body {
                font-family: Work Sans, Arial, Helvetica, sans-serif;
                font-size: 16px;
                color:black;
            }

  • O corpo do site usa Work Sans (importada do Google).
  • Caso não esteja disponível, cai para Arial → Helvetica → sans-serif.
  • Tamanho fixo de 16px e cor padrão preta.

Estilização de títulos


            h1 {
                font-family: Honk, serif;
                font-size: 3em;
            }
            h2 {
                font-family: Honk, serif;
                font-size: 2em;
            }

  • Tanto h1 quanto h2 usam a fonte Honk (do Google).
  • O serif é o fallback, caso a fonte falhe.
  • Tamanhos aumentados em relação ao corpo do texto (3em e 2em).

Fontes Locais

Definição de fontes locais:


                @font-face {
                    font-family: 'Asia';
                    src: url('fontes/Hoshiko Satsuki.ttf') format('truetype');
                    font-weight: normal;
                    font-style: normal;
                }

  • @font-face → permite registrar uma fonte personalizada.
  • font-family: 'Asia'; → nome usado para aplicar a fonte no CSS.
  • src: url(...) format(...) → caminho do arquivo local e seu formato (TrueType).
  • Essa regra define a fonte Asia a partir de um arquivo .ttf localizado em fontes/.


            @font-face { 
                font-family: 'ghot';
                src: url('fontes/MorrisRomanBlack.otf') format('opentype'), 
                    url('fontes/MorrisRomanBlack.ttf') format('truetype');
                font-weight: normal;
                font-style: normal;
            }

  • Fonte chamada ghot.
  • Suporta dois formatos: .otf (OpenType) e .ttf (TrueType).
  • Ter mais de uma opção aumenta a compatibilidade entre navegadores.


            .fontalt {
                h2 {
                    font-family: 'Asia', 'Trebuchet MS';
                    font-size: 2em;
                    font-weight: normal;
                    font-style: normal;
                    color: azure;
                }
                p{
                    font-family: 'ghot', 'Impact';
                    font-size: 20px;
                    font-weight: normal;
                    font-style: normal;
                }
            }

  • Dentro da classe .fontalt, são aplicadas fontes alternativas:
    • h2: usa 'Asia' (local), se não existir cai para 'Trebuchet MS'.
      • Texto azul-claro (azure) e tamanho de 2em.
    • p: usa 'ghot' (local), se não existir cai para 'Impact'.
      • Tamanho fixo de 20px.
  • O objetivo é variar a tipografia em partes específicas da página.
