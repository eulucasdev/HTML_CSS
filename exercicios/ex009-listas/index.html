LISTAS EM HTML

Listas Ordenadas

<ol> ⇒ Odered List: Para criar uma lista de forma ordenada.
Ex:

              
        <ol type="I" start="5">
            <li>Acordar
            <li>Ligar para joão
            <li>Tomar café
            <li>Escovar os dentes
        </ol>
  1. Acordar
  2. Ligar para joão
  3. Tomar café
  4. Escovar os dentes

A teg <ol>, tem tipo e inicio, como demonstrado no exemplo.
O type="" podem ser:
1 = númerico - por padrão;
A = alfabeto maiúsculo;
a = alfabeto minúsculo;
I = Romano maiúsculo;
i = Romano minúsculo;

Listas Desordenada

<ul> ⇒ Unordered List: Para criar uma lista não ordenada!
Ex:

        
        <ul type="circle">
            <li>Leite
            <li>Pão
            <li>Arroz
            <li>Feijão
            <li>2kl de peito
            <li>Bolacha
        </ul>
  • Leite
  • Pão
  • Arroz
  • Feijão
  • 2kl de peito
  • Bolacha
