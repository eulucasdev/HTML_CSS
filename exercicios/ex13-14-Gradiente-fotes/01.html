1. Estilização Global e Tipografia Base

O que está acontecendo:


                    * {
                        font-family: Arial, Helvetica, sans-serif;
                    }

  • *: É o seletor universal, que aplica o estilo a todos os elementos da página.
  • font-family: Define a família de fontes que será usada no texto.
  • Arial, Helvetica e sans-serif são fontes de texto. Aqui, o navegador tenta usar a primeira (Arial); se não estiver disponível, tenta a segunda (Helvetica) e por fim uma genérica (sans-serif).


                        body {
                        background-image: linear-gradient(to top, #5c7764 10%, #12a804 );
                        font-size: 1em;
                        font-weight: 500;
                        }

  • background-image: Usa um gradiente linear como imagem de fundo.
  • linear-gradient(to top, ...) cria um degradê de baixo para cima com as cores especificadas.
  • font-size: 1em: Define o tamanho da fonte relativo ao elemento pai (1em geralmente equivale a 16px).
  • font-weight: 500: Define o peso (espessura) da fonte. Valores variam de 100 (fino) até 900 (bem grosso); 500 é semi-negrito.
