3. Estética dos Títulos e Parágrafos

Estilização do <h1>:


                        h1 {
                            color: #CEF09D;
                            text-align: center;
                            text-shadow: 1px 1px 2px #566440ad;
                            font-style: italic;
                            font-size: 2.3em;
                        }

  • color: #CEF09D; → define a cor do texto (verde claro).
  • text-align: center; → centraliza o texto horizontalmente.
  • text-shadow: 1px 1px 2px #566440ad; → sombra no texto: deslocamento 1px horizontal, 1px vertical, 2px de desfoque.
  • cor semitransparente (#566440ad).
  • font-style: italic; → texto em itálico.
  • font-size: 2.3em; → tamanho relativo, maior que o padrão.

Estilização do <h2>:


                    h2 {
                        color: #B8ECD7;
                        text-shadow: 1px 1px 2px #124d35c4;
                        font: oblique bolder 1.8em Times New Roman, sans-serif; 
                    }

  • color: #B8ECD7; → cor verde-azulado suave.
  • text-shadow: 1px 1px 2px #124d35c4; → sombra mais escura para contraste.
  • font: oblique bolder 1.8em Times New Roman, sans-serif;
    • (Essa é a forma shorthand (abreviada) de escrever a fonte:)
    • oblique = estilo inclinado.
    • bolder = peso mais forte que bold.
    • 1.8em = tamanho da fonte.
    • Times New Roman = família preferida.
    • sans-serif = fallback genérico.
