2. Layout e Estilização do Conteúdo Principal


                    main {
                    background-color: rgba(255, 255, 255, 0.438);
                    border-radius: 10px;
                    box-shadow: 5px 5px 15px #083f038f;
                    width: 600px;
                    padding: 10px;
                    margin: auto;
                    }

  • background-color: → Define a cor de fundo.
  • rgba(255, 255, 255, 0.438) → representa uma cor branca translúcida (o "a" é de alpha, que define a transparência).
  • border-radius: 10px: → Arredonda os cantos da caixa com 10 pixels de raio.
  • box-shadow: → Cria uma sombra ao redor da caixa:
  • 5px 5px 15px #083f038f → sombra deslocada 5px para direita, 5px para baixo, com desfoque de 15px e cor esverdeada transparente.
  • width: 600px: → Define a largura da caixa como 600 pixels.
  • padding: 10px: → Espaço interno entre o conteúdo e a borda da caixa.
  • margin: auto: → Centraliza o elemento na horizontal.
